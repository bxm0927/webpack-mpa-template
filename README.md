# webpack-mpa-template

这是一个使用 webpack 配置多页应用(MPA)的简易 demo。

## Framework

主要框架和库的版本：

Node.js 10.x | webpack 4.x | babel 7.x | babel-loader 8.x

## Usage

```bash
# install dependencies
$ npm install

# dev
$ npm run dev

# build
$ npm run build

# analyse
$ npm run analyse
```
